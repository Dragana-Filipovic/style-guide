Notification

Notifications are presented to users to indicate that an background action came to an end. The used animations are a key element for this component since it should get the users attention.

</script>

UX Checklist

  • Use notifications to indicate the state of background actions like copying to the clipboard.
  • This component guarantees the attention of the user. Don't overuse it.
  • It is not to be used as a confirmation page.
