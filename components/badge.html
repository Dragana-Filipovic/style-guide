Badge

A badge is an element which can be attached to other components as an indicator for new or unread items.

See code
HTML
<div class="button">Button with a badge<span class="button__badge badge">159</span></div>

UX Checklist

  • If a badge is placed, the underlying element must be clickable.
  • A badge always only contains a number. No icons allowed.
  • It is always placed in the upper right corner of an element.
