Creating custom icons

Find some advise for creating custom icons below

  • Size the view box as large as the icon really is (do not add extra space to the top/right/bottom/left).
  • Remove any extraneous markup added by your editor. We recommend using SVGO to remove this markup.
  • For brand icons (e.g. social media icons) include the fill property in he SVG itself.
  • Do NOT add a fill property in a non-brand icon (e.g. a download icon).
  • To prevent blur effects through sub-pixel rendering you should design your icons in the size(es) you'll use it on your page. (You can use media queries within SVGs to have diffrent looks for an icon depending on its size. Note that for SVGs the width and height of the view box is used for media queries and not of the browser viewport.)
