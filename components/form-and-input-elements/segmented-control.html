Segmented control

A segmented control enables the user to make a single selection from just a few options.

Basic Segmented Control

This element aligns different options next to each other.

See code
HTML
JavaScript

The segmented-control--js modifier will add custom styling to radio buttons to look like a segmented-control. The segmented-control plugin does that automatically. It also sets the .is-active state on the active item.

<div class="l-container">
  <div class="form">
    <div class="form__group">
      <label class="form__group__label">
        <div class="form__group__label__text">Form label</div>
      </label>
      <div class="form__group__control">
        <div data-segmented-control="data-segmented-control" class="segmented-control">
          <label class="segmented-control__item">
            <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">Ja</span>
          </label>
          <label class="segmented-control__item">
            <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">Nein</span>
          </label>
        </div>
      </div>
    </div>
  </div>
</div>

If you can't use the jQuery plugin, use the segmented-control--js modifier for styling. Set the .is-active state on the correct segmented-control__item and be sure to handle keyboard navigation.

UX Checklist

  • Default values can be set if appropriate - if in doubt, do not set defaults.

  • Do not use more than 5 options in a segmented control. A dropdown can be an alternative.

Stacked Segmented Control

A stacked segmented control is used when text gets to long to be presented next to each other. If you are using the Segmented Control jQuery Plugin, this will be automatically handled for you.

See code
HTML

Add the segmented-control--stacked modifier to create stacked segmented-controls. The jQuery plugin will do that automatically if there's not enough space.

<div style="max-width:200px;">
  <div data-segmented-control="data-segmented-control" class="segmented-control">
    <label class="segmented-control__item">
      <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">Foo bar</span>
    </label>
    <label class="segmented-control__item">
      <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">long text</span>
    </label>
    <label class="segmented-control__item">
      <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">and some</span>
    </label>
    <label class="segmented-control__item">
      <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">more controls</span>
    </label>
  </div>
</div>

UX Checklist

  • Do not set a default value when using a stacked segmented control.

Disabled Segmented Control

A disabled segmented control can be used to show a value that can't be modified.

See code
HTML

Add the disabled="disabled" property to the segmented-control to make it disabled.

<div class="l-container">
  <div class="form">
    <div class="form__group">
      <label class="form__group__label">
        <div class="form__group__label__text">Form label</div>
      </label>
      <div class="form__group__control">
        <div data-segmented-control="data-segmented-control" disabled="disabled" class="segmented-control">
          <label class="segmented-control__item">
            <input type="radio" name="bar" checked="checked" class="segmented-control__item__radio"/><span class="segmented-control__item__label">Ja</span>
          </label>
          <label class="segmented-control__item">
            <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">Nein</span>
          </label>
        </div>
      </div>
    </div>
  </div>
</div>

Backup: Radio Buttons

If presented with problems, the user should always be able to use the browser native radio buttons as a fallback solution.

HTML

Use the same markup to create standard radio buttons. Just omit the jQuery plugin and the segmented-control--js modifier.

        <div class="form__group__label__text">Form label</div>

          </label>
          <label class="segmented-control__item">
            <input type="radio" name="bar" class="segmented-control__item__radio"/><span class="segmented-control__item__label">Nein</span>
          </label>
        </div>
      </div>
    </div>
  </div>
</div>
